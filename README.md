# **Folders Manager**

## How to use

### Download or clone this repo

```
 git clone https://github.com/kriz1618/foldersManager
 cd foldersManager
```

### Install

```
npm run start
```

Modify the entry in the file entryFile.txt

```
Example
    CREATE fruits/mongose
    CREATE vegetables
    CREATE fruits/carrot
    MOVE fruits/carrot vegetables
    LIST
```

### Run

```
npm start
```

### Run tests

```
npm run test
```

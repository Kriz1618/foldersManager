# **Folders Manager**

## How to use

Download or clone this repo

```
 git clone
 cd foldersManager
```

Modify the params in the line *121*

```
Example
const arg = [
    'CREATE fruits/mongose',
    'CREATE vegetables',
    'CREATE fruits/carrot',
    'MOVE fruits/carrot vegetables',
    'LIST'
```

Run

```
npm index.js
```
